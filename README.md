# Social Media Performance Dataset – README

## Overview

In today’s digital-first landscape, social media performance provides valuable insights into audience behavior, content effectiveness, and brand growth.

This dataset captures key engagement and reach metrics from social media platforms, making it suitable for data analysis, dashboard development, performance optimization studies, and business intelligence projects.

It can be used by beginners learning analytics as well as professionals building insight-driven dashboards and reports.

---

## Dataset Description

The dataset contains structured social media performance data with the following key variables:

Post_ID – Unique identifier for each post
Date – Date of content publication
Platform – Social media platform (e.g., Instagram, Facebook, Twitter)
Content_Type – Type of content (Image, Video, Reel, Carousel, etc.)
Likes – Total number of likes received
Comments – Total number of comments received
Shares – Total number of shares
Reach – Number of unique users who saw the content
Impressions – Total number of times the content was displayed
Engagement_Rate – Engagement relative to reach or impressions

## Key Metrics Explanation

Engagement Rate is typically calculated as:

(Total Engagements / Reach) × 100

or

(Total Engagements / Impressions) × 100

Where:
Total Engagements = Likes + Comments + Shares
